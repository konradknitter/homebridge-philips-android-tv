Instructions

This application pairs with Philips Android TV, providing API credentials essential for the Homebridge plugin.

Go to:
  1. TV Settings
  2. Network
  3. Wired network or Wi-Fi
  4. Show network settings
On the TV screen you will see two pieces of information required to be provided to the plugin:
  • IPv4 Address of TV. Necessary for the pairing process and all later interactions with TV.
  • MAC Address (Wired or Wireless). Necessary for Wake over LAN features. (Turning the TV on from deep sleep).
On the same level in Settings as "Show network settings", you should find WoWLAN settings - make sure that it's enabled if you want to use the Wake over Wireless LAN feature.

Pairing

To start the pairing process, be in front of the TV, and provide a TV's IPv4 Address.
